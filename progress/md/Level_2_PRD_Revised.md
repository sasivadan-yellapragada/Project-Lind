# Level 2 PRD - Revised

This revised PRD is intentionally kept aligned with `progress/md/Level_2_PRD.md`.

The current Level 2 scope, implemented architecture, API contract, decision rationale, and validation note are documented in:

- `progress/md/Level_2_PRD.md`
- root `README.md`

Key corrections from the earlier draft:

- The implemented backend is Express/Node, not FastAPI.
- The implemented databases are SQLite `trials.db` and `user_data.db`, not MySQL.
- `trials.db` is a prerequisite source artifact generated by Level 1.
- `GET /api/trials` returns `{ "data": [...], "meta": {...} }`.
- Reviewer data is stored in a separate SQLite file, so `nct_id` integrity is application-enforced rather than database-foreign-key-enforced.
- Watchlist, notes, and tags persistence should be validated by add -> refresh/restart -> retrieve.
- The exploratory `scraper/` artifacts are not part of L2 and have been removed.
